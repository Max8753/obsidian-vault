HTTP = Hypertext Transfer Protocol
Ein Protokoll das bestimmt wie wir Daten, die in HTML (Hypertext Markup Language) geschrieben wurden, austauschen können. 

HTTP-Requests können per GET oder POST übertragen werden
GET: Daten stehen im Header = Unsicher
POST: Daten stehen im Body = Sicher

HTTP Ist der Standard im Web. 

Gute Ressource:
https://developer.mozilla.org/de/docs/Web/HTTP/Reference/Status
