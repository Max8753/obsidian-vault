
JSON = JavaScript Object Notation

Der Standard für Web-API´s (API = Application Programming Interface - Programmierschnittstelle)

Vorteil:
- Schlank, wenig Zeilen. Lässt sich leicht lesen
- Lässt sich leicht und schnell verarbeiten

Beispiel:

```
{

  

    "benutzer": {

        "id": 101,

        "vorname": "Harry",

        "nachname": "Potter",

        "email": "harry@hogwarts.com",

        "status": "aktiver Schüler",

        "hobbys": ["Programmieren", "Quidditch"]

    }

  

}
```
